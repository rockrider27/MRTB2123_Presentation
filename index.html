Section 2 · Analytical Diagnosis

Four Structural Limitations

01

Data Fragmentation

No unified customer ID. Data distributed across disconnected systems. Prevents 360° profiling — degrades all downstream analytics.

02

Descriptive Analytics Only

Anchored at "what happened" dashboards. No predictive churn models or CLV estimation. Decisions are reactive, not proactive.

03

Weak Behavioural Segmentation

Segments defined by GMV or login frequency only. Cannot distinguish promotion-dependent from brand-loyal customers.

04

Weak Recommendation System

Rule-based logic and historical data only. Repetitive suggestions. Forgoes the data network effect that TikTok Shop exploits.

Vicious Cycle: Data fragmentation degrades segmentation → poor segmentation makes predictions unreliable → organisation reverts to descriptive-only → weak recommendations starve the analytical layers of signal.

Section 3 · Analytics Framework

Four-Layer Architecture

L1

Foundation — Unified Customer Data Platform

Single persistent customer ID unifying transactional, browsing, campaign, and demographic data. Real-time segmentation. Prerequisite for every layer above.

L2

Descriptive — Behavioural Metric Dashboards

Individual-level signals: browse-to-purchase ratio, discount-redemption rate, category dwell time, cart-abandonment frequency, session recency.

L3

Predictive — Clustering · Churn · CLV

K-Means behavioural segmentation · XGBoost/LightGBM churn scoring · BG/NBD + Gamma-Gamma CLV estimation per customer and segment.

L4

Prescriptive — Hybrid Recommendations + Next-Best-Action

Collaborative + content-based + real-time signals + serendipity injection. Next-best-action determines personalised treatment per segment.
